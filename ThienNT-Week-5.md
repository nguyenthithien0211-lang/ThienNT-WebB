## Git and Gemini Learning Summary

### 1. Git: Distributed Version Control System
Git is a fundamental tool for tracking changes in source code during software development, ensuring data integrity and supporting non-linear workflows.

* **Definition:** A distributed version control system (VCS) designed to handle everything from small to very large projects with speed and efficiency.
* **Core Functionalities:**
    * **Version Control:** Uses `git commit` to create snapshots of your project, allowing you to track history and revert to previous states.
    * **Collaboration:** Enables multiple developers to work on the same codebase simultaneously without overwriting each other's changes.
    * **Branching & Merging:** Facilitates the creation of independent lines of development (branches) to test new features safely before merging them into the main project.
    * **Remote Repositories:** Syncs local work with platforms like GitHub or GitLab for cloud backup and team accessibility.

### 2. Gemini: Multimodal Generative AI
Gemini is Google's state-of-the-art artificial intelligence, built to be natively multimodal and capable of reasoning across different types of information.

* **Definition:** A high-performance AI model designed to understand, operate across, and combine different types of information including text, code, audio, image, and video.
* **Core Functionalities:**
    * **Content Generation & Translation:** Drafts high-quality text and provides accurate translations across numerous languages, including English, Korean, and Vietnamese.
    * **Coding & Debugging:** Assists in writing complex code in languages like Java and Python, as well as identifying and fixing logical bugs.
    * **Multimodal Processing:** Analyzes and generates creative assets, from detailed images to video summaries based on user prompts.
    * **Advanced Reasoning:** Provides insightful answers to complex technical queries by processing vast amounts of information in real-time.
