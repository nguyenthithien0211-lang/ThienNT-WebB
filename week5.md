# Learning Summary Report: GitHub, Gemini & Markdown

## 1. Markdown (.md) - The Essential Markup Language
Markdown is a crucial tool for quickly creating documentation (like README files) and organizing study notes.

* **Headings:** Use `#` (The more `#` you use, the smaller the heading gets).
* **Text Formatting:** * `**Bold**` to emphasize important keywords.
    * `*Italics*` for side notes or descriptions.
* **Lists:** Use `-` or `*` for bullet points.
* **Tables (Tables):** Use vertical bars `|` and hyphens `-` to align data cleanly.
* **Links:** Use the `[Display Text](URL)` format to insert external reference links.

---

## 2. GitHub - Source Code & Version Control
GitHub is the ultimate platform for hosting coding projects and tracking development history.

* **Repository (Repo):** The storage space holding all project files and their revision history.
* **GitHub.dev:** A quick web-based code editor accessed by simply pressing `.` (period) while viewing a repository.
* **The Commit Workflow:**
    1.  **Edit:** Make necessary changes to your code or text files.
    2.  **Commit Message:** Write a concise description of what was changed (e.g., `docs: update learning notes`).
    3.  **Commit & Push:** Save the changes and push them to the remote server to secure your progress.
* **Commit Hash:** A unique ID generated for every commit (often used by instructors to verify on-time assignment submissions).

---

## 3. Gemini - Multimodal AI Assistant
Gemini serves as an intelligent companion to boost productivity in both studying and working.

* **Coding Support:** Troubleshoots bugs, generates sample code (for Java, Python, SQL, etc.), and optimizes programming logic.
* **Language Learning:** Provides natural translations and explains grammar concepts (useful for English TOEIC or Korean practice).
* **Multimodal Processing:** Goes beyond text by analyzing images (like assignment screenshots) and understanding complex prompts.
* **Project Ideation:** Helps brainstorm and structure personal projects and automation workflows.

---

## 4. Conclusion
Combining these three tools creates a highly professional developer workflow:
1.  Use **Gemini** to research concepts and overcome technical blockers.
2.  Use **Markdown** to document your knowledge clearly and structurally.
3.  Use **GitHub** to back up your notes, share your work, and prove your learning progress through a solid commit history.
